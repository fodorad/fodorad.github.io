Personality Questionnaires

BFI-2 Code

Motivation

This repository offers a suite of tools designed to simplify the processing of personality questionnaires in research experiments. The motivation behind this project stems from the need for a standardized and automated approach to handle questionnaire data efficiently across various experimental settings. By providing a robust package with support for different questionnaires and export formats, researchers can focus more on experimental design and analysis, rather than spending time on manual data processing.

Supported Questionnaires

Name Status Description
BFI-2 Supported The initial focus of the repository is on supporting the BFI-2 questionnaire, a widely used instrument for assessing personality traits based on the Big Five framework. The BFI-2 provides valuable insights into an individual's personality across dimensions such as openness, conscientiousness, extraversion, agreeableness, and neuroticism. The package facilitates the processing of BFI-2 questionnaire responses, offering scaled outputs representing both Big Five values and FACET scores, which can be utilized in various research projects.
HEXACO Soon! The HEXACO model extends the Big Five framework by incorporating an additional factor, honesty-humility, providing a more comprehensive understanding of personality traits.
VAS-F Soon! On the other hand, The Visual Analogue Scale for Fatigue Severity (VAS-F) offers a succinct means of assessing fatigue levels before and after experimental tasks. With 18 questions prompting participants to circle numbers between 0 and 10, the pre-experiment assessment provides insight into baseline fatigue levels, while the post-experiment version captures changes resulting from the task. Quick to complete, both assessments take less than one minute, offering researchers valuable data on participant fatigue and its response to experimental manipulations.

By incorporating support for these additional questionnaires, the repository seeks to cater to a broader range of research needs.

API and Export Formats

This repository can be easily installed using pip, making it simple to get started. One of the key features of the repository is its flexible API, which allows seamless integration into various experimental settings. Researchers can easily incorporate the questionnaire processing functionality into their workflows, whether they are conducting online surveys or laboratory experiments.

Additionally, the package supports exporting questionnaire data in multiple formats, including numpy ndarrays and CSV files, ensuring compatibility with different analysis tools and software environments.
